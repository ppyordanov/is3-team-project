is3-team-project
================

University of Glasgow, Interactive Systems 3 group project

  A project relating to the indentification of relationships and pattern recognition between two large data sources - a WHO file with general country-based statistical information and an Olympic Medal table.

Team members: Toni Andreev, Tsvetan Yordanov, Mihail Tachev, Silvia Sotirova, Peter Yordanov
